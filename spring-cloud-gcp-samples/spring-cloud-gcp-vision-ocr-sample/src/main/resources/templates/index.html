This application demonstrates how one can upload documents to Google Cloud Storage and easily run OCR analysis on the documents. This application will allow you to upload documents to your bucket, run OCR analysis on them, and then retrieve the results.

Process Documents

You may upload a document to your GCS bucket: [[${ocrBucket}]]. Then this app runs OCR analysis on the document and allows you to view the parsed text.
